# shaedy RCON

A CounterStrikeSharp plugin that lets admins run server commands from in-game chat.

## Features

- Execute any server command via chat
- Requires `@css/root` permission
- Logs executed commands to the console

## Commands

| Command | Description | Permission |
|---------|-------------|------------|
| `!rcon <command>` | Execute a server command | `@css/root` |

## Installation

Drop the plugin folder into your CounterStrikeSharp `plugins` directory.

## Configuration

No config needed. Permission is handled via CounterStrikeSharp's admin system.
